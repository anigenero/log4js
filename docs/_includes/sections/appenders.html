Appenders

Console

Default appender that outputs logs to the system out

{% highlight typescript %} configure({ layout : '%d [%p] %c %M:%line:%column - %m %ex', appenders: [{ appender: 'Console' }] }); {% endhighlight %}

File

A simple file appender that outputs logs to a single file.

{% highlight typescript %} configure({ layout : '%d [%p] %c %M:%line:%column - %m %ex', appenders: [{ appender: 'File', config: { fileName: './logs/app.log' } }] }); {% endhighlight %}

Rolling File

The rolling file appender allows for logs to be appended to a log file, to be split and backed up once the file reaches a certain size.

{% highlight typescript %} configure({ layout : '%d [%p] %c %M:%line:%column - %m %ex', appenders: [{ appender: 'RollingFile', config: { fileName: './logs/rollingfile.log', filePattern: './logs/rollingfile.%i.log', maxBackup: 10, maxSize: 10 } }] }); {% endhighlight %} {% include sections/custom-appender.html %}

Pattern Layout

log4js2 borrows tags from the Log4j2 library

Logger

%c %logger

Date

%d{DEFAULT} %d{ABSOLUTE} %d{COMPACT} %d{DATE} %d{ISO8601} %d{ISO8601_BASIC}

Exception

%ex %exception %throwable

File

%F %file

Map

%K %map %MAP

Line

%L %line

Column

%column

Log Message

%m %msg %message

Method

%M %method

Line Separator

%n

Log Level

%p %level

Relative

%r %relative

Sequence

%sn %sequenceNumber
